# BALF Exercise Pack

This folder is fully separate from the rest of the project and contains practice exercises for:
- DRY
- SRP
- YAGNI
- Mockito basics

## Files
- exercises/01_dry_basics.md
- exercises/02_dry_refactor_java.java
- exercises/03_srp_analysis.md
- exercises/04_yagni_user_story.java
- exercises/05_mockito_controller_test.java
- solutions/answer_key.md

## How to use
1. Solve the tasks in `exercises/` first.
2. Compare with `solutions/answer_key.md` afterwards.
3. Repeat until you can explain each principle in your own words.
